Regeln

Regeln für den Chernarus.eu Server

Hinweis zu den Regeln:

Der verstoß gegen unsere Regeln kann zum Bann von den Chernarus.eu Server führen. Regeln nicht gelesen ist keine Ausrede da ihr ingame darauf aufmerksam gemacht werdet. Es besteht kein Recht auf das spielen auf Chernarus.eu

Benehmen:

DayZ ist ein Spiel...zugegeben eins das mit vielen Emotionen verbunden ist...aber es bleibt ein Spiel. Wir dulden keine Politik, Religion, Sexismus, Rassismus, Beleidigungen, äußerung über das Alter und Geschlecht oder anderen Shit Talk.

Player vs Environment:

Chernarus.eu ist ein PvE Server. Das bedeutet das dass töten von Spielern verboten ist.

Raiden von Basen:

Das Raiden von Basen ist verboten. Unter einer Base versteht man auf Chernarus.eu nur Basen oder Gebäube die verschlossen sind. Alles andere ist keine Base und kann geplündert werden. Also gibt euch ein wenig Mühe eure Sachen zu sichern.

Bauen von Basen:

Das Bauen einer base ist durch den Mod @Simple Base oder durch das Vanilla System möglich. Das bauen innerhalb von Gebäuden die als wichtige Plätze für Loot angesehen werden ist verboten. Dazu zählen alle Orte mit Military, Medic und Police Loot. Ebenso dürfen keine Brunnen zugebaut werden.

Bilden von Loot Haufen:

Das bilden von Loot Haufen oder das lagern in einen Fass oder sonstiges um neuen Loot zu generieren ist verboten.
